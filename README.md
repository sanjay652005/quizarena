## 🚀 Features

- 🎮 **Real-time multiplayer** — compete with multiple players simultaneously via Socket.IO
- 🏠 **Room-based sessions** — create or join quiz rooms instantly with a room code
- 🏆 **Live leaderboard** — real-time score updates after every question
- 🤖 **AI-generated questions** — dynamic quiz content powered by AI
- 🔐 **JWT Authentication** — secure login and protected routes
- 📱 **Responsive UI** — works seamlessly across all devices

## ⚡ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/sanjay652005/quizarena.git
cd quizarena
```

### 2. Install dependencies

```bash
# Backend
cd backend
npm install

# Frontend
cd ../frontend
npm install
```

### 3. Configure environment variables

Create `.env` in `backend/`:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
PORT=5000
```

Create `.env` in `frontend/`:

```env
VITE_API_URL=https://quizarena-uima.onrender.com
```

### 4. Run the project

```bash
# Backend (from /backend)
npm run dev

# Frontend (from /frontend)
npm run dev
```

## 🎮 Core Modules

### Room System
Players create a quiz room and share the room code with friends. Multiple players can join the same room and compete in real time.

### Real-time Gameplay
Built with Socket.IO — every question, answer, and score update is synced instantly across all players in the room.

### Live Leaderboard
After each question, the leaderboard updates live showing rankings, scores, and correct answers for all players.

### AI Question Generation
Questions are dynamically generated by AI based on selected topics, ensuring a fresh quiz experience every time.
