# QuizArena 🎯

Real-Time Multiplayer Quiz Platform built using the MERN Stack.

## 🚀 Features

* Real-time multiplayer quiz rooms
* AI-generated quiz questions
* Live leaderboard
* User authentication
* Responsive UI
* Socket.io based real-time updates

## 🛠️ Tech Stack

* React.js
* Node.js
* Express.js
* MongoDB
* Socket.io
* Tailwind CSS

## ⚙️ Installation

```bash
git clone https://github.com/sanjay652005/QuizArena.git

cd QuizArena

npm install
```

Run backend:

```bash
cd backend
npm run dev
```

Run frontend:

```bash
cd frontend
npm run dev
```

## 🎮 Functionalities

* Create Quiz Room
* Join Quiz Room
* Real-Time Gameplay
* Live Score Tracking
* AI Question Generation
* Multiplayer Competition
